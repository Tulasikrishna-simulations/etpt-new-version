## Why two sites instead of one updated one

The 2024 portfolio was built during a drop year as the work's first formalization. It is preserved as-is because **iteration is itself the credential**. A researcher with one portfolio shows they had an idea. A researcher with portfolio v1 and a separate program site shows the work moved forward, deepened, and developed across time. Both sites stay live.

The 2024 portfolio tells the autobiography (panda → Comfort Trap → Mirror Rule → four archetypes → Tsukuba vision). The 2026 site is operations: the blind perception thesis, six derived equations, the simulator, the 36-month dossier, the partnership ask.
